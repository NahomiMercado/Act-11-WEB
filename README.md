# Menus Project

This project was developed in Laravel and includes a main menu with three options:

- Home
- Photos
- Contact

## Features

- A different view for each menu option
- Active menu highlighting depending on the current section
- Shared footer displayed on all views
- Routes and controller to resolve each view

## Technologies used

- Laravel
- Blade
- PHP
- HTML/CSS
